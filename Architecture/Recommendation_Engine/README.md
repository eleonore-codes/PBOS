# Recommendation Engine

## Purpose

The Recommendation Engine is the decision intelligence layer of PBOS.

It transforms PBHS findings, capability scores, bottlenecks, opportunities and constraints into prioritized executive recommendations.

PBHS answers:

> Where am I?

The Recommendation Engine answers:

> What should I do next?

---

# Mission

The Recommendation Engine helps the business owner decide where to invest the next unit of Human Time, money, attention or AI capacity to create the greatest Business Return and Life Return according to their Vision of Life.

---

# Core Function

The Recommendation Engine receives:

- PBHS capability profile
- evidence confidence
- bottleneck analysis
- opportunity analysis
- Vision of Life constraints
- Human Signature priorities
- available AI Employees
- DIY / DWY / DFY options

It produces:

- ranked recommendations
- expected Business Return
- expected Life Return
- confidence levels
- suggested execution path
- AI Employee assignments
- success criteria
