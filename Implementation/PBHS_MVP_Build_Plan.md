# PBHS MVP Build Plan

## Purpose

Define the minimum build required to make PBHS usable by Customer Zero.

---

# MVP Goal

A Business Owner can:

1. Create a business profile.
2. Complete the PBHS assessment.
3. Receive capability scores.
4. Receive prioritized recommendations.
5. Read an executive report.

---

# Build Scope

## Included

- Business profile
- PBHS questionnaire
- scoring engine
- recommendation generation
- executive report
- basic dashboard

---

## Excluded

- full Executive Board simulation
- Operational AI Employees
- departments
- strategy workspace
- payments
- multi-user teams
- external integrations
